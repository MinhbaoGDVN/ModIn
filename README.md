# 📖 About

**ModIn** is a lightweight Minecraft Modpack Manager for **Minecraft Java Edition**.

Instead of manually copying dozens of `.jar` files every time you switch projects, ModIn lets you organize your mods into separate Modpacks and switch between them in seconds.

---

# ✨ Features

- 📦 Create Modpacks
- 📂 List Modpacks
- 🔄 Switch Between Modpacks
- 🔍 Duplicate Detection
- ⚡ Lightweight
- 🪶 Portable
- 🖥 Windows Native
- 🌍 Open Source

---

# ⚙️ How It Works

```text
Modpack A
      │
      ▼
.minecraft/mods
      ▲
      │
Modpack B
```

ModIn automatically copies the selected Modpack into your Minecraft `mods` folder.

---

# 📥 Installation

1. Download **ModInSetup.exe** from the latest Release.
2. Install ModIn.
3. Launch ModIn.
4. Create your Modpacks.
5. Enjoy!
